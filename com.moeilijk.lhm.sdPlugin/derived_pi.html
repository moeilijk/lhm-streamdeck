Formula
Formula reference

Sum — adds all slot values together.
A + B + C + …
Example: fan 1 + fan 2 + fan 3 = total airflow.

Average — sum divided by number of slots.
(A + B + C + …) ÷ n
Example: average of all core temperatures.

Max — the highest value across all slots.
max(A, B, C, …)
Example: hottest core in the CPU.

Min — the lowest value across all slots.
min(A, B, C, …)
Example: coolest core in the CPU.

Delta — second slot minus first. Fixed at 2 slots.
B − A
Example: outlet temp − intake temp = heat added.

Percent — first slot as share of the rest.
A ÷ (B + C + …) × 100
Example: used RAM ÷ total RAM × 100 = usage %.
